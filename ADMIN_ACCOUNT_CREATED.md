## 🆘 Troubleshooting

### Issue: Can't login

**Solutions:**
1. Verify email: `admin@chefio.app`
2. Verify password: `ChefioAdmin2026!`
3. Check you're at `/admin/login` not `/login`
4. Clear browser cache
5. Try incognito mode

### Issue: Redirected to regular login

**Solutions:**
1. Verify you're using admin login URL
2. Check Firestore document has `role: "admin"`
3. Check Firestore document has `isAdmin: true`
4. Logout and login again

### Issue: Access denied

**Solutions:**
1. Verify admin fields in Firestore
2. Check AdminRoute component
3. Review browser console for errors
4. Verify Firebase rules allow admin access

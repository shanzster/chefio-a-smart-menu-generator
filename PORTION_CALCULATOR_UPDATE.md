## 🔧 Technical Implementation

### Save to Firebase
```javascript
await saveRecipeToUser(user.uid, {
  name: `${recipeName} (${servings} servings)`,
  servings: desiredServings,
  ingredients: scaledIngredients,
  nutrition: scaledNutrition,
  scaledFrom: {
    originalRecipeId: id,
    originalServings: original,
    scaleFactor: factor
  }
});
```

### Generate PNG
```javascript
// Create canvas
const canvas = document.createElement('canvas');
const ctx = canvas.getContext('2d');

// Draw header, ingredients, checkboxes
// Convert to blob
canvas.toBlob((blob) => {
  // Download as PNG
  const url = URL.createObjectURL(blob);
  link.download = 'shopping-list.png';
  link.click();
});
```
